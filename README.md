# MustGym

**MustGym** is an elite, all-in-one gym management platform developed by **Regulus Labs**, designed to provide a seamless, secure, and data-driven experience for gym owners, fitness professionals, and members.

## Comprehensive Feature Ecosystem

MustGym provides a unified environment for both gym operations and athlete development:

### 🏋️ Athletes & Members
- **Interactive Dashboard:** Real-time visibility of gym capacity, hydration tracking, and personal records.
- **Elite Gamification:** Points system (XP), levels, and virtual badges to reward consistency.
- **Fitness Challenges:** Community-wide challenges with integrated logging and leaderboards.
- **AI-Powered Services:** Custom workout routines and Indian diet plans tailored by Gemini AI.
- **QR Check-in:** Seamless entry/exit scanning for automated session logging.

### 🏛️ Administration & Operations
- **Command Center:** Real-time member management, equipment inventory, and maintenance logs.
- **Financial Ledger:** Integrated payment tracking and revenue analytics (INR).
- **AI Business Insights:** Predictive analytics and at-risk member identification via Gemini AI.
- **Announcement Hub:** Centralized system to push critical updates and news.
- **Tiered Memberships:** Define and manage Basic, Standard, and Elite membership architecture.

For a full breakdown of every module, refer to [APP_FEATURES.md](./APP_FEATURES.md).

## Tech Stack

MustGym is built with a modern, high-performance tech stack:

- **Frontend:** React (TypeScript) with Tailwind CSS for adaptive, polished user interfaces.
- **Backend:** Node.js/Express for robust API management.
- **Database & Auth:** Firebase (Firestore & Firebase Authentication).
- **Security:** Zero-trust, Attribute-Based Access Control (ABAC) Firestore Security Rules.

## Deployment & Development

This project is structured for easy development and deployment using modern cloud infrastructure.

1. **Development:** Use `npm run dev` to start the application with Vite hot module replacement in a local or containerized development environment.
2. **Build:** Execute `npm run build` to generate the production-ready build artifacts.
